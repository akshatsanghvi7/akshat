Daikibo Data Analytics Internship

Data Analytics Virtual Internship: Task 1 solved in Tableau, Task 2 solved in Excel.

Back to Projects

Project Overview

This internship involved two tasks for Daikibo Manufacturing:

  • Task 1 (Tableau): Analyzed May 2021 telemetry data from four factories (Tokyo, Osaka, Berlin, Shenzhen) to identify the factory with the highest failure rate and the most failure-prone machine types, delivering interactive Tableau dashboards.
  • Task 2 (Excel): Classified compensation equality scores into Fair (-10 to +10), Unfair (<-10 or >10), and Highly Discriminative (<-20 or >20) categories using Excel formulas.

Key Features

  • Cross-Location Failure Analysis: Unified data from four global factories.
  • High-Frequency Telemetry Monitoring: Detailed 10-minute interval insights.
  • Machine-Type Breakdown: Pinpointed top failure contributors.
  • Compensation Equality Audit: Evaluated pay fairness across roles.
  • Automated Classification: Rule-based Excel logic for transparency.
  • Interactive Dashboards: Real-time exploration in Tableau.

Challenges & Solutions

  • Data Complexity: Flattened and preprocessed JSON for Tableau ingestion.
  • Volume Handling: Leveraged Tableau filters and time-series charts to manage high-frequency data.
  • Threshold Definition: Implemented nested IF formulas in Excel for clear classification.
